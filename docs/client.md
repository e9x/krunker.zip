# Using Sketch in a client

You can use Sketch in a Krunker client by injecting it using the [Electron Injector](https://injector.krunker.zip/).

### Usage

The injector can be found at https://injector.krunker.zip/.

Follow this tutorial for injecting a script with a URL: https://github.com/e9x/electron-injector/tree/master#injecting-a-script-with-a-url

Sketch Script URL (copy and paste into the "Script URLs" tab of the injector):
<https://krunker.zip/stable/sketch.user.js>
