# Menu Settings

This documentation provides an overview of the recommended settings for the menu feature in the game.

## Menu Key

The menu key is a customizable key binding that opens and closes the in-game menu when pressed. To set the menu key, click on the "Menu Key" button and press the desired key. You can also reset or unbind the menu key.

## Menu Button

This setting determines whether the menu button is displayed on the game screen or not. Note that disabling the menu button requires a game restart. To enable or disable the menu button, toggle the switch labeled "Menu Button".

## Silent Updates/Key

This setting is recommended for streamers. When enabled, the cheat will silently fail if there's an update, the access key expires, or the cheat isn't updated. This option requires the player to manually check for updates to the cheat, ensure it's updated, and that the key isn't expired. If any of these occur, the cheat won't load, and the player won't be able to re-enable this option without reinstalling the cheat. To enable or disable silent updates/key, toggle the switch labeled "Silent Updates/Key".
