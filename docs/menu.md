# Menu Settings

This documentation provides an overview of the recommended settings for the menu feature in the game.

## Menu Key

The menu key is a customizable key binding that opens and closes the in-game menu when pressed. To set the menu key, click on the "Menu Key" button and press the desired key. You can also reset or unbind the menu key.

## Menu Button

This setting determines whether the menu button is displayed on the game screen or not. Note that disabling the menu button requires a game restart. To enable or disable the menu button, toggle the switch labeled "Menu Button".

## Hide Updates/Key

The "Hide Updates/Key" setting is recommended for streamers. When enabled, the cheat will silently fail if there's an update, the access key expires, or the cheat isn't updated. This helps maintain discretion while streaming, but it also means you'll need to manually disable the setting in case of updates or expired keys.

### Website

This is if you're using a web browser and can go to links like the one below.

To disable the "Hide Updates/Key" setting, click on the following link: [https://krunker.io/#showUpdates](https://krunker.io/#showUpdates). This will reset the setting and allow updates and key notifications to show again.

### Desktop Client

You'll have to manually modify your Sketch config file.

On Windows:

1. Enable hidden files [(Tutorial)](https://support.microsoft.com/en-us/windows/view-hidden-files-and-folders-in-windows-97fbc472-c603-9d90-91d0-1166d1d9f4b5)
2. Open explorer and enter in the address bar:
   `%USERPROFILE%`
3. Find and delete `.photoshop.sketch`
