# Skill Settings

This documentation provides an overview of the recommended settings for the skill features in the game. These features include bhop, slidehop, wall jump, and force auto.

## Bhop

When enabled, holding the spacebar allows players to continuously jump, or "bhop." To enable or disable bhop, toggle the switch labeled "Bhop".

## Slidehop

When enabled, holding the crouch key allows players to slidehop. To enable or disable slidehop, toggle the switch labeled "Slidehop".

## Wall Jump

When enabled, bhop will also automatically jump on walls. Note that bhop must be enabled for this setting to work. To enable or disable wall jump, toggle the switch labeled "Wall Jump".

## Force Auto

When enabled, non-automatic guns will become automatic, allowing for continuous fire. To enable or disable force auto, toggle the switch labeled "Force Auto".

# Triggerbot Settings

This section provides an overview of the recommended settings for the triggerbot feature in the game.

## Triggerbot Key

The triggerbot key is a customizable key binding that activates the triggerbot when pressed. To set the triggerbot key, click on the "Triggerbot Key" button and press the desired key. You can also reset or unbind the triggerbot key.

## Triggerbot

When enabled and the right-click is held, the triggerbot feature automatically shoots enemies that come into the player's line of sight. To enable or disable triggerbot, toggle the switch labeled "Triggerbot".

## Triggerbot Minimum (Seconds)

This setting determines the minimum delay between triggerbot shots, measured in seconds. Use the slider labeled "Triggerbot Minimum (Seconds)" to adjust the value between 0 and 1, with 0.05 increments.

This section provides an overview of the recommended settings for the recoil control feature in the game.

## Recoil Control Key

The recoil control key is a customizable key binding that activates the recoil control feature when pressed. To set the recoil control key, click on the "Recoil Control Key" button and press the desired key. You can also reset or unbind the recoil control key.

## Recoil Control

When enabled, the recoil control feature reduces the recoil effect on weapons, allowing for more accurate shots. To enable or disable recoil control, toggle the switch labeled "Recoil Control".

## Recoil Smooth Factor

This setting controls the speed of the aimbot's rotation, affecting how quickly the weapon aims while compensating for recoil. Use the slider labeled "Recoil Smooth Factor" to adjust the value between 0 and 1, with 0.05 increments.
